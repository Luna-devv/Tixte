Troubleshooting


Tixte displays that it has not detected the DNS change!!

The custom domains can take up to 48h to register, but usually takes 2-4 hours, if it doesn't register after 48h, try checking if your domain is set up properly, and if you're using Cloudflare, try setting your domain SSL to "Flexible", and if nothing works then you probably haven't set it up correctly.
