Tixteregister


1.Go to tixte.com/register.

2.Enter a username, Email, password and a beta code.

3.If you want to use ShareX or MagicCap head over to the configurations tab and download the config.

4.If you want to link your Tixte account, go into #bot-commands and type /link TIXTE_UPLOAD_KEY
(Upload Key can be found in the configurations tab)




Tixtebeta code


Enter a giveaway

To get notified about giveaways, go into #information and get the Giveaway role and whenever the developers host a giveaway, you can get the chance of getting a code!



Boost the server

By boosting the server, @Tixte Manager will send you a DM with a beta access code, as well as Tixte Turbo , so if you boost, make sure your DMs are open!



Apply via the form

Go to tixte.cc/beta and enter your email, you should get an email when the developers start giving out codes. Please know, applying for a beta code doesn't mean you'll get one.



Ask friendly in the chat

If you join our discord server you can ask nicely in #general if someone has a beta code left, if someone is so nice he will give you one.
