Tixteplaceholders


{{size.UNIT}} file size - replace UNIT with b (byte), kb (kilobyte), mb (megabyte), or gb (gigabyte).

{{author.created_at}} timestamp of when the file owners tixte account was created.

{{author.username}} file owners tixte username.

{{domain}} file domain (i.e: cdn.xyna.space).

{{extension}} file extension (i.e: png).

{{uploaded_at}} file creation date.

{{size}} file size - auto formatted.

{{author.id}} file owners tixte id.

{{id}} file id.




Advancedplaceholders soon™️


{{random.string.MIN_LENGTH-MAX_LENGTH}} random string of characters, length between the MIN_LENGTH and MAX_LENGTH.

{{random.number.MIN_NUMBER-MAX_NUMBER}} random number between the MIN_LENGTH and MAX_LENGTH.

{{option1|option2|option3}} choose a random option for each file upload between the {{}}, separated with |.

{{random.color}} returns a random HEX value.
